# Ready Sprites

โฟลเดอร์นี้ใช้เก็บ sprite ที่ผ่านการตัดเฟรม จัด baseline และพร้อมนำเข้า Godot แล้ว

โครงสร้างในแต่ละตัวละคร:

```text
frames/      # PNG frame แยก
sheets/      # sprite sheet ที่จัด grid แล้ว
resources/   # Godot resource เช่น SpriteFrames .tres
```

อย่าใส่ภาพ AI ดิบไว้ในโฟลเดอร์นี้ ให้เก็บภาพดิบใน `_ai_raw/` ก่อน
