# AI Raw Assets

โฟลเดอร์นี้ใช้เก็บภาพต้นฉบับที่สร้างจาก AI เท่านั้น

ไฟล์ในนี้ยังไม่ถือว่าเป็น asset พร้อมใช้จริงใน Godot เพราะอาจมีปัญหา:

- frame ไม่ตรง grid
- baseline ไม่เท่ากัน
- ตัวละครเลื่อนตำแหน่งระหว่างเฟรม
- ดาบหรือมือเพี้ยน
- ภาพยังไม่ได้ตัดพื้นหลัง

เมื่อจัดเฟรมเสร็จแล้ว ให้ย้ายไฟล์ที่พร้อมใช้ไปไว้ใน `assets/sprites/`, `assets/ui/`, หรือ `assets/backgrounds/`
