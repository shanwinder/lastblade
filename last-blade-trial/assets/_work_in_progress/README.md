# Work In Progress Assets

ใช้โฟลเดอร์นี้สำหรับไฟล์ที่กำลังตัดเฟรม จัด baseline หรือทดสอบ scale ก่อนย้ายไปยังโฟลเดอร์พร้อมใช้จริง

ตัวอย่างงานที่เหมาะกับโฟลเดอร์นี้:

- ภาพที่กำลังลบพื้นหลัง
- frame ที่ยังจัดตำแหน่งไม่เสร็จ
- sprite sheet ที่กำลังทดสอบ grid
- export ชั่วคราวจาก Aseprite, Krita, Photoshop หรือเครื่องมืออื่น
